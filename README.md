# Aidoku-Extension-And-Source-Repo
The serves as a unified repository for all 3rd party extensions for Aidoku

All credit given to the developers of these extensions. I will keep updating this repo whenever I find new ones. Some repos may have sources the main repo doesn't so you can check for yourself and see. 

**Please leave any issues on the developer's repo or discord**

## Installing an extension
- Install Aidoku
- Tap on the github.io link below
- Tap "Add to Aidoku"

# Contributing
Please fork, add the extension to all areas where it is applicable, and submit a pull request. 

If the extension has both comic and manga sources, then add it to both.
